# Senior Python Developer - FinTech Innovation

## Company: TechFin Solutions
*Leading financial technology company revolutionizing digital payments*

## About the Role
We're looking for a Senior Python Developer to join our fast-growing engineering team. You'll be responsible for designing, building, and maintaining our core payment processing systems that handle millions of transactions daily.

## Key Responsibilities
- Design and implement scalable backend services using Python and FastAPI
- Develop and maintain microservices architecture
- Optimize application performance and ensure high availability
- Collaborate with cross-functional teams to define, design, and ship new features
- Implement automated testing and continuous integration/continuous deployment (CI/CD) pipelines
- Mentor junior developers and participate in code reviews

## Essential Criteria
- 5+ years of professional Python development experience
- Strong experience with FastAPI or similar Python web frameworks
- Solid understanding of RESTful API design principles
- Experience with cloud platforms (AWS, GCP, or Azure)
- Proficient with SQL and NoSQL databases
- Experience with containerization (Docker, Kubernetes)
- Strong problem-solving skills and attention to detail
- Excellent communication and teamwork abilities

## Desirable Criteria
- Experience in fintech or payment processing
- Knowledge of event-driven architectures
- Experience with message brokers (Kafka, RabbitMQ)
- Familiarity with machine learning concepts
- Contributions to open-source projects

## Technical Stack
- **Backend:** Python 3.9+, FastAPI, Django
- **Databases:** PostgreSQL, MongoDB, Redis
- **Cloud:** AWS (EC2, S3, Lambda, RDS)
- **DevOps:** Docker, Kubernetes, GitHub Actions
- **Testing:** pytest, unittest, Postman
